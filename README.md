# leet.vim
Implements leetspeak converter at Vimscript<br>
This is created to learn Vimscript.



## Setup and Usage

1. In terminal

```bash
git clone https://github.com/shinshin86/leet.vim.git
cd leet.vim
vim
```

2. In vim

```vim
" 1. read vim file
:source plugin/leet.vim

" 2. insert leet text
:Leet hello
```
